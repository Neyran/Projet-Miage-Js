
# Projet-Miage-Js
Nous comptons réaliser un projet autour du mastermind , Le projet consistera à retrouver un code défini aléatoirement par le programme . Nous comptons nous inspirer de l'intégralité des règles de base du mastermind (affichage de pastilles de couleur / 12 essaies pour trouver la combinaison de base) Le joueur serait toujours dans le rôle de décoder le code couleur . 
Les modules ne sont actuellement pas complètement défini mais voici une liste des modules actuellement défini : 
-Module d'encodage du code couleur
-Module de gestion des pastilles rouge et blanche (servant à aider le joueur trouver la bonne combinaison) 
-Module permettant au joueur d'établir des combinaisons
-Module de gestion de la difficulté(exemple : réduction du nombre d'essais initial / augmentation du nombre de couleur au sein de la combinaison ext...)
